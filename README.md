# GitIntel AI Analysis

AI 驱动的 GitHub 仓库智能分析工具，基于 Next.js + Tailwind CSS + Zustand 构建。

## 技术栈

- **框架**: Next.js 15 (App Router)
- **样式**: Tailwind CSS v4
- **状态管理**: Zustand
- **动画**: Framer Motion
- **图表**: Recharts
- **图标**: Lucide React

## 快速开始

```bash
npm install
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看效果。
